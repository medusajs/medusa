## updateCollections(selector, data, sharedContext?): Promise&#60;[ProductCollectionDTO](../../../interfaces/product.ProductCollectionDTO/page.mdx)[]&#62;

```ts
const collections =
  await productModuleService.updateCollections(
    {
      id: ["pcol_123", "pcol_321"],
    },
    {
      title: "Summer Collection",
    }
  )
```
