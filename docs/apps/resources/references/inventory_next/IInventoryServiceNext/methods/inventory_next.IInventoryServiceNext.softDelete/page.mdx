# softDelete - Inventory Next Module Reference

This documentation provides a reference to the `softDelete` method. This belongs to the Inventory Next Module.

This method soft deletes inventory items by their IDs.

## Example

```ts
await inventoryModuleService.softDelete(
  ["iitem_123", "iitem_321"],
  {
    returnLinkableKeys: ["location_level"],
  }
)
```
