# create - Region Module Reference

This documentation provides a reference to the `create` method. This belongs to the Region Module.

## create(data, sharedContext?): Promise&#60;[RegionDTO](../../../interfaces/region.RegionDTO/page.mdx)[]&#62;

This method creates regions.

### Example

```ts
const region = await regionModuleService.create([
  {
    name: "Europe",
    currency_code: "eur",
    countries: ["dk", "de", "fr"],
  },
  {
    name: "United States of America",
    currency_code: "usd",
    countries: ["us"],
  },
])
```
