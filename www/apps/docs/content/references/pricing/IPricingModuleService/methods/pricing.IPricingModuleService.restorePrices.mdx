# restorePrices - Pricing Module Reference

This documentation provides a reference to the `restorePrices` method. This belongs to the Pricing Module.

This method restores soft deleted prices by their IDs.

## Example

```ts
await pricingModuleService.restorePrices([
  "price_123",
  "price_321",
])
```
