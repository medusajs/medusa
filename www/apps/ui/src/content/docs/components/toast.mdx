## Usage

---

<br />
Add the Toaster component somewhere in your tree

<Snippet>```import {Toaster} from "@medusajs/ui"```</Snippet>

<Snippet>
  ```
  <Toaster />
  ```
</Snippet>

The `useToast` hook returns a `toast` function that you can use to display a toast.

<Snippet>```import {useToast} from "@medusajs/ui"```</Snippet>

<Snippet>
```
const { toast } = useToast()

return (

  <Button 
    onClick={() => 
      toast({ 
        title: "Toast title",
        description: "Toast body",
      })
    }
  >
    Trigger
  </Button>
)
```
</Snippet>

## API Reference

This component is based on the [Radix UI Toast](https://www.radix-ui.com/primitives/docs/components/toast) primitives.
