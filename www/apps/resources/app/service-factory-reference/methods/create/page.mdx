# {metadata.title}

This method creates one or more records of the data model.

## Create One Record

```ts
const post = await customModuleService.createPosts({
  name: "My Post",
  published_at: new Date(),
  metadata: {
     external_id: "1234"
  }
})
```

If an object is passed of the method, an object of the returned record is also returned.

---

## Create Multiple Records

```ts
const posts = await customModuleService.createPosts([
  {
    name: "My Post",
    published_at: new Date()
  },
  {
    name: "My Other Post",
    published_at: new Date()
  }
])
```

If an array is passed of the method, an array of the created records is also returned.
