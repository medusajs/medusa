# revertUsage - Promotion Module Reference

This documentation provides a reference to the `revertUsage` method. This belongs to the Promotion Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method is used to revert the changes made by registerUsage action

## Example

```ts
await promotionModuleService.revertUsage([
  {
    amount: 50,
    code: "50OFF",
  },
  {
    amount: 5000,
    code: "FREESHIPPING",
  },
])
```
