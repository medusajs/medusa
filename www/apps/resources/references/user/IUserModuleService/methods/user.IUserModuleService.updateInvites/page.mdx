# updateInvites - User Module Reference

This documentation provides a reference to the `updateInvites` method. This belongs to the User Module.

## updateInvites(data, sharedContext?): Promise&#60;[InviteDTO](../../../interfaces/user.InviteDTO/page.mdx)[]&#62;

This method updates existing invites.

### Example

```ts
const invites = await userModuleService.updateInvites([
  {
    id: "invite_123",
    accepted: true
  },
  {
    id: "invite_321",
    metadata: {
      test: true
    }
  }
])
```
