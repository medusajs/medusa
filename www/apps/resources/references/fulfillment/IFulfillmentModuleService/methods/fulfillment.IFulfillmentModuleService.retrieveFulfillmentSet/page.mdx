# retrieveFulfillmentSet - Fulfillment Module Reference

This documentation provides a reference to the `retrieveFulfillmentSet` method. This belongs to the Fulfillment Module.

This method retrieves a fulfillment set by its ID.

## Example

A simple example that retrieves a fulfillment set by its ID:

```ts
const fulfillmentSet = await fulfillmentModuleService.retrieveFulfillmentSet(
  "fuset_123"
)
```

To specify relations that should be retrieved:

```ts
const fulfillmentSet = await fulfillmentModuleService.retrieveFulfillmentSet(
  "fuset_123",
  {
    relations: ["service_zones"],
  }
)
```
