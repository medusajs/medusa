# softDeleteFulfillmentSets - Fulfillment Module Reference

This documentation provides a reference to the `softDeleteFulfillmentSets` method. This belongs to the Fulfillment Module.

This method soft deletes fulfillment sets by their IDs.

## Example

```ts
await fulfillmentModuleService.softDeleteFulfillmentSets([
  "fuset_123",
  "fuset_321",
])
```
