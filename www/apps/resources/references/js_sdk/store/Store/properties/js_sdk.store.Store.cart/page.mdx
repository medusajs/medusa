# cart - JS SDK Store Reference

This documentation provides a reference to the `sdk.store.cart` set of methods used to send requests to Medusa's Store API routes.

Related guides: [How to implement carts in the storefront](https://docs.medusajs.com/v2/resources/storefront-development/cart).

## create

This method creates a cart. It sends a request to the [Create Cart](https://docs.medusajs.com/v2/api/store#carts\_postcarts)
API route.

Related guide: [How to create a cart in the storefront](https://docs.medusajs.com/v2/resources/storefront-development/cart/create).

### Example

```ts
sdk.store.cart.create({
  region_id: "reg_123"
})
.then(({ cart }) => {
  console.log(cart)
})
```
