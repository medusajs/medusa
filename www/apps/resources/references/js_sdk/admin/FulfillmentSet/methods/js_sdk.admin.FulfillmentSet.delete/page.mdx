# delete

This method deletes a fulfillment set. It sends a request to the
[Delete Fulfillment Set](https://docs.medusajs.com/api/admin#fulfillment-sets\_deletefulfillmentsetsid)
API route.

## Example

```ts
sdk.admin.fulfillmentSet.delete("fset_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
