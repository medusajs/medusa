# batchCustomers

This method manages customers of a group to add or remove them from the group.
It sends a request to the [Manage Customers](https://docs.medusajs.com/api/admin#customer-groups\_postcustomergroupsidcustomers)
API route.

## Example

```ts
sdk.admin.customerGroup.batchCustomers("cusgroup_123", {
  add: ["cus_123"],
  remove: ["cus_321"]
})
.then(({ customer_group }) => {
  console.log(customer_group)
})
```
