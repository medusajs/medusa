# delete

This method deletes a customer group. This method sends a request to the
[Delete Customer Group](https://docs.medusajs.com/api/admin#customer-groups\_deletecustomergroupsid)
API route.

## Example

```ts
sdk.admin.customerGroup.delete("cusgroup_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
