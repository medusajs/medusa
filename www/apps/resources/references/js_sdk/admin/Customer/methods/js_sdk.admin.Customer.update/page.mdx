# update

This method updates a customer's details. It sends a request to the
[Update Customer](https://docs.medusajs.com/api/admin#customers\_postcustomersid) API route.

## Example

```ts
sdk.admin.customer.update("cus_123", {
  first_name: "John"
})
.then(({ customer }) => {
  console.log(customer)
})
```
