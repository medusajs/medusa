# delete

This method deletes a campaign by its ID. It sends a request to the
[Delete Campaign](https://docs.medusajs.com/api/admin#campaigns\_deletecampaignsid) API route.

## Example

```ts
sdk.admin.campaign.delete("procamp_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
