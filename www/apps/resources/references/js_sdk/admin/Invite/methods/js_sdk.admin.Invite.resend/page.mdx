# resend

This method refreshes the token of an invite. It sends a request to the
[Refresh Invite Token](https://docs.medusajs.com/api/admin#invites\_postinvitesidresend)
API route.

## Example

```ts
sdk.admin.invite.resend("invite_123")
.then(({ invite }) => {
  console.log(invite)
})
```
