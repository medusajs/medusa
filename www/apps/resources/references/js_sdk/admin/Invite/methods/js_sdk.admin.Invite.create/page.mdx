# create

This method creates an invite. It sends a request to the
[Create Invite](https://docs.medusajs.com/api/admin#invites\_postinvites)
API route.

## Example

```ts
sdk.admin.invite.create({
  email: "user@gmail.com",
})
.then(({ invite }) => {
  console.log(invite)
})
```
