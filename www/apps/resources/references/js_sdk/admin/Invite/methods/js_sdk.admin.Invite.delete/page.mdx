# delete

This method deletes an invite. It sends a request to the
[Delete Invite](https://docs.medusajs.com/api/admin#invites\_deleteinvitesid)
API route.

## Example

```ts
sdk.admin.invite.delete("invite_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
