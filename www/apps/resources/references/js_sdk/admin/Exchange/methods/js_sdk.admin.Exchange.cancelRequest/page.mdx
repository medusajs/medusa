# cancelRequest

This method cancels an exchange request. It sends a request to the
[Cancel Exchange Request](https://docs.medusajs.com/v2/api/admin#exchanges\_deleteexchangesidrequest)
API route.

## Example

```ts
sdk.admin.exchange.cancel("exchange_123")
.then(({ deleted }) => {
  console.log(deleted)
})
```
