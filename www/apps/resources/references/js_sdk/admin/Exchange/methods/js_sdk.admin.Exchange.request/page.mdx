# request

This method confirms an exchange request, applying its changes on the associated order.

This method sends a request to the [Confirm Exchange](https://docs.medusajs.com/v2/api/admin#exchanges\_postexchangesidrequest)
API route.

## Example

```ts
sdk.admin.exchange.request("exchange_123", {})
.then(({ exchange }) => {
  console.log(exchange)
})
```
