# create

This method creates a claim. It sends a request to the 
[Create Claim](https://docs.medusajs.com/api/admin#claims\_postclaims) API route.

## Example

```ts
sdk.admin.claim.create({
  type: "refund",
  order_id: "order_123",
})
.then(({ claim }) => {
  console.log(claim)
})
```
