# addItems

This method adds items to the claim. It sends a request to the
[Add Items](https://docs.medusajs.com/v2/api/admin#claims\_postclaimsidclaimitems) API route.

## Example

```ts
sdk.admin.claim.addItems("claim_123", {
  items: [
    {
      id: "orli_123",
      quantity: 1
    }
  ]
})
.then(({ claim }) => {
  console.log(claim)
})
```
