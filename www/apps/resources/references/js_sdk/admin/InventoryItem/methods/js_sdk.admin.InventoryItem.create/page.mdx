# create

This method creates an inventory item. It sends a request to the 
[Create Inventory Item](https://docs.medusajs.com/v2/api/admin#inventory-items\_postinventoryitems)
API route.

## Example

```ts
sdk.admin.inventoryItem.create({
  sku: "SHIRT"
})
.then(({ inventory_item }) => {
  console.log(inventory_item)
})
```
