# batchUpdateLevels

This method manages the inventory levels of an inventory item. It sends a request to the
[Manage Inventory Levels](https://docs.medusajs.com/v2/api/admin#inventory-items\_postinventoryitemsidlocationlevelsbatch)
API route.

## Example

```ts
sdk.admin.inventoryItem.batchUpdateLevels("iitem_123", {
  create: [{
    location_id: "sloc_123",
    stocked_quantity: 10
  }],
  delete: ["sloc_123"]
})
.then(({ inventory_item }) => {
  console.log(inventory_item)
})
```
