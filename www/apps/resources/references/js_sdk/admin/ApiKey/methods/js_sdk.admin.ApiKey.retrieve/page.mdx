# retrieve

This method retrieves an API key's details. It sends a request to the 
[Get API key](https://docs.medusajs.com/api/admin#api-keys\_getapikeysid) API route.

## Example

```ts
sdk.admin.apiKey.retrieve("apk_123")
.then(({ api_key }) => {
  console.log(api_key)
})
```
