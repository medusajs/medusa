# batchSalesChannels

This method manages the sales channels associated with a publishable API key to either add
or remove associations. It sends a request to the [Manage Sales Channels](https://docs.medusajs.com/api/admin#api-keys\_postapikeysidsaleschannels)
API route.

## Example

```ts
sdk.admin.apiKey.batchSalesChannels("apk_123", {
  add: ["sc_123"],
  remove: ["sc_321"]
})
.then(({ api_key }) => {
  console.log(api_key)
})
```
