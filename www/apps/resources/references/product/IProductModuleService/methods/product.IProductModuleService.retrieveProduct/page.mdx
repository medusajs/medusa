# retrieveProduct - Product Module Reference

This documentation provides a reference to the `retrieveProduct` method. This belongs to the Product Module.

This method is used to retrieve a product by its ID

## Example

A simple example that retrieves a product by its ID:

```ts
const product = await productModuleService.retrieveProduct("prod_123")
```

To specify relations that should be retrieved:

```ts
const product = await productModuleService.retrieveProduct("prod_123", {
  relations: ["categories"],
})
```
