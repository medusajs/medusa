# createProductOptions - Product Module Reference

This documentation provides a reference to the `createProductOptions` method. This belongs to the Product Module.

## createProductOptions(data, sharedContext?): Promise&#60;[ProductOptionDTO](../../../interfaces/product.ProductOptionDTO/page.mdx)[]&#62;

This method is used to create product options.

### Example

```ts
const options = await productModuleService.createProductOptions([
  {
    title: "Color",
    values: ["Blue", "Green"],
    product_id: "prod_123",
  },
  {
    title: "Size",
    values: ["Small", "Medium"],
    product_id: "prod_321",
  },
])
```
