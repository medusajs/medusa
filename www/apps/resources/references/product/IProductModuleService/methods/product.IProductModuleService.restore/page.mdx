# restore - Product Module Reference

This documentation provides a reference to the `restore` method. This belongs to the Product Module.

This method is used to restore products which were deleted using the [softDelete](../product.IProductModuleService.softDelete/page.mdx) method.

## Example

```ts
await productModuleService.restore(["prod_123", "prod_321"])
```
