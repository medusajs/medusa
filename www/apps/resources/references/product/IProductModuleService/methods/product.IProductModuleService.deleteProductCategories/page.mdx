# deleteProductCategories - Product Module Reference

This documentation provides a reference to the `deleteProductCategories` method. This belongs to the Product Module.

This method is used to delete categories by their ID.

## Example

```ts
await productModuleService.deleteProductCategories([
  "pcat_123",
  "pcat_321",
])
```
