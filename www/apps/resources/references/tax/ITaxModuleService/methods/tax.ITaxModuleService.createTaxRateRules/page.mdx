# createTaxRateRules - Tax Module Reference

This documentation provides a reference to the `createTaxRateRules` method. This belongs to the Tax Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## createTaxRateRules(data, sharedContext?): Promise&#60;[TaxRateRuleDTO](../../../interfaces/tax.TaxRateRuleDTO/page.mdx)&#62;

This method creates a tax rate rule.

### Example

```ts
const taxRateRule = await taxModuleService.createTaxRateRules(
  {
    reference: "product_type",
    reference_id: "ptyp_123",
    tax_rate_id: "txr_123",
  }
)
```
