# restoreTaxRates - Tax Module Reference

This documentation provides a reference to the `restoreTaxRates` method. This belongs to the Tax Module.

This method restores soft deleted tax rates by their IDs.

## Example

```ts
await taxModuleService.restoreTaxRates(["txr_123", "txr_321"])
```
