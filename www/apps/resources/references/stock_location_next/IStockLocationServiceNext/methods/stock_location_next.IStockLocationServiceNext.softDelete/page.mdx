# softDelete - Stock Location Next Module Reference

This documentation provides a reference to the `softDelete` method. This belongs to the Stock Location Next Module.

This method soft deletes stock locations by their IDs.

## Example

```ts
await stockLocationModuleService.softDelete([
  "sloc_123",
  "sloc_321",
])
```
