# restoreStockLocations - Stock Location Next Module Reference

This documentation provides a reference to the `restoreStockLocations` method. This belongs to the Stock Location Next Module.

This method restores soft deleted stock locations by their IDs.

## Example

```ts
await stockLocationModuleService.restoreStockLocations([
  "sloc_123",
  "sloc_321",
])
```
