# restoreInventoryItems - Inventory Next Module Reference

This documentation provides a reference to the `restoreInventoryItems` method. This belongs to the Inventory Next Module.

This method restores soft deleted inventory items by their IDs.

## Example

```ts
await inventoryModuleService.restoreInventoryItems(
  ["iitem_123", "iitem_321"],
  {
    returnLinkableKeys: ["location_level"],
  }
)
```
