# retrieveReservationItem - Inventory Next Module Reference

This documentation provides a reference to the `retrieveReservationItem` method. This belongs to the Inventory Next Module.

This method retrieves a reservation item by its ID.

## Example

A simple example that retrieves a reservation item by its ID:

```ts
const reservationItem =
  await inventoryModuleService.retrieveReservationItem(
    "resitem"
  )
```

To specify relations that should be retrieved:

```ts
const reservationItem =
  await inventoryModuleService.retrieveReservationItem(
    "resitem",
    {
      relations: ["inventory_item"],
    }
  )
```
