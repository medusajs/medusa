# default Method - API Reference

This method defines the default value of a property.

## Example

```ts
import { model } from "@medusajs/utils"

const MyCustom = model.define("my_custom", {
  color: model
    .enum(["black", "white"])
    .default("black"),
  age: model
    .number()
    .default(0),
  // ...
})

export default MyCustom
```
