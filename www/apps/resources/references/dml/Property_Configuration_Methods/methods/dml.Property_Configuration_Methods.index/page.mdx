# index Method - API Reference

This method defines an index on a property.

## Example

```ts
import { model } from "@medusajs/utils"

const MyCustom = model.define("my_custom", {
  id: model.id(),
  name: model.text().index(
    "IDX_MY_CUSTOM_NAME"
  ),
})

export default MyCustom
```
