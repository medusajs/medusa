# updateReturnReasons - Order Module Reference

This documentation provides a reference to the `updateReturnReasons` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## updateReturnReasons(selector, data, sharedContext?): Promise&#60;[OrderReturnReasonDTO](../../../interfaces/order.OrderReturnReasonDTO/page.mdx)[]&#62;

This method updates existing return reasons matching the specified filters.

### Example

```ts
const returnReasons = await orderModuleService.updateReturnReasons({
  id: "123"
}, {
  label: "Damaged"
})
```
