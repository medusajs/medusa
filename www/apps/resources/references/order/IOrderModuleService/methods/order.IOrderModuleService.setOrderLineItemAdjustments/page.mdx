# setOrderLineItemAdjustments - Order Module Reference

This documentation provides a reference to the `setOrderLineItemAdjustments` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method sets the line item adjustments of an order.

## Example

```ts
const lineItemAdjustments = await orderModuleService.setOrderLineItemAdjustments(
  "123",
  [
    {
      item_id: "1234",
      amount: 10
    },
    {
      id: "123",
      item_id: "4321",
      amount: 20
    }
  ]
)
```
