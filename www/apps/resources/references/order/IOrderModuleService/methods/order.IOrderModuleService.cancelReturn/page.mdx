# cancelReturn - Order Module Reference

This documentation provides a reference to the `cancelReturn` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method cancels an order return.

## Example

```ts
const orderReturn = await orderModuleService.cancelReturn({
  return_id: "123"
})
```
