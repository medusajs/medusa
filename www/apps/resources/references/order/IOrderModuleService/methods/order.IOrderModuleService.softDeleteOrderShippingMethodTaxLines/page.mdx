# softDeleteOrderShippingMethodTaxLines - Order Module Reference

This documentation provides a reference to the `softDeleteOrderShippingMethodTaxLines` method. This belongs to the Order Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method soft deletes shipping method tax lines by their IDs.

## Example

```ts
await orderModuleService.softDeleteOrderShippingMethodTaxLines([
  "123", "321"
])
```
