# deleteRefundReasons - Payment Module Reference

This documentation provides a reference to the `deleteRefundReasons` method. This belongs to the Payment Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

## deleteRefundReasons(refundReasonId, sharedContext?): Promise&#60;void&#62;

This method deletes a refund reason by its ID.

### Example

```ts
await paymentModuleService.deleteRefundReasons([
  "refr_123",
  "refr_321",
])
```
